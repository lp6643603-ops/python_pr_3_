# 🎓 Student Data Organizer

A Python-based console application to manage student records efficiently. This project demonstrates the practical use of Python data structures such as Lists, Dictionaries, Tuples, and Sets along with CRUD (Create, Read, Update, Delete) operations.

---

## 📌 Features

✅ Add New Student Records

✅ Display All Student Records

✅ Update Student Information
- Update Age
- Update Subjects

✅ Delete Student Records

✅ Display Unique Subjects Offered

✅ String Formatting Demonstration
- f-Strings
- format() Method
- % Formatting

## 📂 Data Structure Usage

### 📋 List
Used to store multiple student records.

```python
students = []
```

### 📖 Dictionary
Used to store student details.

```python
student = {
    "student_info": student_info,
    "name": name,
    "age": age,
    "grade": grade,
    "subjects": list(subjects)
}
```

### 📌 Tuple
Stores immutable student information.

```python
student_info = (student_id, dob)
```

### 🎯 Set
Ensures unique subjects without duplicates.

```python
subjects = set()
```

## 📝 Student Details Stored

- Subjects

```python

    "name": "Rahul",
    "age": 18,
    "grade": "12th",
    "subjects": ["Math", "Science", "English"]
}
```
